# Frontend - Sistema de Gestión de Préstamo de Aulas DTIC

Este es el frontend de la aplicación distribuida para la gestión y reporte de préstamos de aulas, desarrollado con React.

## Instalación

1. Ingresa a la carpeta `frontend`
2. Ejecuta:

npm install

## Ejecución en desarrollo

npm start


## Estructura del proyecto

- `src/`: Código fuente principal
- `src/components/`: Componentes reutilizables de la interfaz
- `src/pages/`: Vistas principales
- `src/services/`: Cliente para APIs REST
- `src/hooks/`: Hooks personalizados
- `src/styles/`: Estilos globales o módulos CSS

---

Este frontend consume las APIs REST de los microservicios de backend (Java/Spring Boot).

## Despliegue

El despliegue se realiza mediante Docker. Consulta el manual de usuario para instrucciones detalladas.
